3D Model Viewer

Upload your own files or configure a gallery of pre-loaded models.

Controls:
• Left drag: Rotate view
• Right drag / Shift+drag: Pan model
• Mouse wheel: Zoom in/out
• Touch: Drag to rotate, pinch to zoom
• Long touch: Pan model
Gallery Setup:
Create models/gallery.js with your model list
Optional: models/models.json to override JS file
Works both locally (with web server) and on GitHub Pages
